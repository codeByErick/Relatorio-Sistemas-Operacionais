Anteriormente, Realizamos a instalação do Arch Linux juntamente com o GNOME Desktop Environment, que é uma interface gráfica para o Linux, também conhecida como GUI (Graphical User Interface), também tivemos um primeiro contato com a linha de comando ou CLI (Command-Line Interface), onde realizamos a instalação do Arch Linux. Agora prosseguiremos falando um pouco mais detalhadamente sobre CLI e GUI, e o que podemos fazer com estes. Além disso, mostraremos exemplos práticos de como navegar entre diretórios e criar arquivos utilizando tanto o terminal quanto a interface gráfica, evidenciando como essas duas formas de interação se complementam dentro do sistema.

Para começar, vá na gaveta de apps e procure por "terminal", você pode notar um outro programa semelhante chamado "Console", também é a linha de comando, mas o console é o acesso direto em modo texto, enquanto o terminal é um emulador gráfico que permite usar o shell dentro do ambiente de desktop. Usar o terminal nos oferece algumas vantagens como:

- Permite abrir várias abas e janelas ao mesmo tempo.

- Facilita copiar e colar comandos ou resultados com o mouse/atalhos.

- Dá para ajustar cores, transparência, tamanho da fonte e até dividir a tela, etc.
